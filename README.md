# Progress Tracker — Assignment Submission

A simple and interactive progress-tracking UI built using React.  
The application displays a step-by-step progress bar, final score page, and a playful UI element (paw icon) with smooth animations.

---

## Setup Instructions

Follow the steps below to run the project locally:

1. **Clone the repository**
   ```bash
   git clone [https://github.com/Nishtha-8104/Wedding_Company_Task](https://github.com/Nishtha-8104/Wedding_Company_Task.git)
   ```

2. **Navigate into the project**
   ```bash
   cd Wedding_Company_Task
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm start
   ```

5. Open in browser at:
   ```
   https://wedding-company-task.vercel.app/
   ```

##  Tech Stack Used

- **React.js** — UI rendering  
- **CSS3 / Tailwind / Custom CSS** — Styling & animations  
- **JavaScript (ES6+)** — Logic and component behavior  
- **Node.js + npm** — Dependency & build management  

---

##  Key Features Implemented

- Multi-step Progress Bar  
- Final Score Page  
- Animated Paw Icon  
- Responsive UI  
- Modular Component Architecture  

---

##  Assumptions Made

- Only frontend UI required  
- Linear step progression  
- No backend or persistent storage  
- Paw icon is decorative  
- Score passed via props/state
